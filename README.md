# Expense Tracker 💰

A command-line Expense Tracker built using Python, Object-Oriented Programming (OOP), JSON, and File Handling.

## Features 🚀

* Add new expenses
* View all expenses
* Search expenses by category
* Delete expenses using Expense ID
* Calculate total expenses
* Generate category-wise expense reports
* Automatic Expense ID generation
* Persistent data storage using JSON
* Input validation and exception handling

## Technologies Used 🛠️

* Python
* OOP (Classes & Methods)
* JSON
* File Handling
* Datetime Module
* Exception Handling

## Project Structure 📂

* `project.py` → Main application file
* `expensetracker.json` → Stores expense data

## How to Run ▶️

1. Clone the repository:

```bash
git clone <repository-link>
```

2. Move into the project directory:

```bash
cd python-expense-tracker
```

3. Run the application:

```bash
python project.py
```
